🎂 Age Calculator

A clean, responsive, and lightweight "Age Calculator" built with HTML, Tailwind CSS, and Vanilla JavaScript. It calculates the exact age in years, months, and days between a date of birth and a selected calculation date.

🌐 Live Demo

🔗 https://mdmoniruzzamanshihab.github.io/Age-Calculation/



📖 Overview

This project allows users to calculate their exact age by selecting a Date of Birth and an optional Calculation Date. If no calculation date is selected, the current date is used automatically. The application features a modern, responsive interface with simple and efficient calculation logic.


✨ Features

 🎂 Calculate exact age in **Years, Months, and Days**
 📅 Supports custom calculation dates
 📆 Uses today's date by default
 📱 Fully responsive design
 ⚡ Instant calculation
 🎨 Modern glassmorphism UI
 💡 Clean and lightweight code
 🚀 No frameworks or build tools required



🛠️ Tech Stack

- HTML5
- Tailwind CSS
- Vanilla JavaScript
